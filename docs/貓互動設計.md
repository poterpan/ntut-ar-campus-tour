## RL 設定(初版,來源:README §5.2)

- **形象**:校園貓
- **遊戲**:AR 餵食小遊戲 —— 使用者點 AR 地面放飼料,貓自主前往覓食
- **觀察空間**:貓位置、最近飼料位置、障礙物、飢餓值
- **動作空間**:連續(方向 + 速度)或離散(前後左右停)—— 待定
- **Reward**:吃到飼料 +10、移動 -0.01、撞牆 -1
- **訓練環境**:Unity Editor 簡化 3D 平面場景,隨機生成飼料與障礙
- **推論**:訓練完 export ONNX,端側用 Sentis 推論
- **訓練機**:RTX 3090 workstation
